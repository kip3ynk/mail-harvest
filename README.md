# MailHarvest

> simple script for mail harvesting

## Instalation

```
$ git clone https://github.com/relarizky/MailHarvest.git
$ cd MailHarvest
$ pip3 install colorama requests tld
$ mkdir -v saved
$ chmod +x main.py
$ ./main.py
```

## Usage
```
$ ./main.py https://target.com/ file_saved.txt
```
